## 📌 Installation :
```console
# Clone the repository
$ git clone https://github.com/4m4Sec/Kharon

# go to the Kharon directory
$ cd Kharon
```  
  
Then, configure your settings in the <a href="https://github.com/4m4Sec/Kharon/blob/main/kharon.py">kharon.py</a> file  
  
```
# Start the script 
$ python3 kharon.py
```
